## React.js

**Comandos para a criação do projeto**
```
Yarn create vite ./ --template react
```
**Chakra-Ui**
```
yarn add @chakra-ui/react @emotion/react @emotion/styled framer-motion @chakra-ui/icons
```
**Roda o projeto**
```
yarn dev
```
